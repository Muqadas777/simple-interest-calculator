# Simple Interest Calculator

A lightweight, web-based tool designed to quickly calculate simple interest based on user inputs for principal amount, annual interest rate, and time period.

## Features
* Simple and clean user interface.
* Instant calculation logic.
* Fully responsive and runs directly in any modern web browser without installation.

## Formula Used
The application calculates interest using the standard formula:
A = P * (1 + r * t)

Where:
* A = Total Accumulation (Principal + Interest)
* P = Principal Amount
* r = Annual Interest Rate (as a decimal)
* t = Time Period (in years)

## How to Run
1. Clone or download this repository to your local machine.
2. Open the `index.html` file in any standard web browser (Chrome, Firefox, Safari, Edge).
3. Enter the required values to see the calculated interest breakdown.
